# Ce Să Fiu?

**Află ce ți se potrivește cu adevărat.**

MVP-ul public al platformei de orientare în carieră pentru elevii români de clasele 9–12.

## Ce este

Un quiz de 10 întrebări care:
- Identifică top 3 direcții de carieră (6 arhetipuri inspirate RIASEC/Holland)
- Sugerează programe universitare reale din România
- Generează un card shareable pentru Story (9:16)

Complet client-side, fără backend, fără cookies, fără tracking. HTML/CSS/JS pur.

## Live

https://chiriac-adrian.github.io/CeSaFiu/
